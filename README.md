# CodeAlpha - Simple E-commerce Store

## Project Description
This is a simple e-commerce website developed as part of the CodeAlpha Full Stack Development Internship.

## Features
- User Registration & Login
- Product Listing
- Product Details
- Shopping Cart
- Order Processing

## Technologies Used
- HTML
- CSS
- JavaScript
- Express.js (or Django)
- MySQL (or MongoDB)

## Installation
1. Clone the repository
2. Install dependencies
3. Run the server
4. Open the project in your browser
